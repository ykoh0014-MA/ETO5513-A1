# About Data
This dataset from Our World in Data website ["Life Satisfaction vs. Life Expectancy"] (https://ourworldindata.org/grapher/life-satisfaction-vs-life-expectancy)  contains information on the life expectancies at birth, population size and self-reported life satisfaction across countries from 2011 to 2024. The original dataset has been reduced to include only selected Asia countries to fit the scope of the assessment.
